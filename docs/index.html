devbox CLI

Manage remote dev boxes from the terminal — provision, connect, sync, and destroy AWS EC2 instances.

Requirements

  • macOS or Linux
  • AWS account

AWS credentials and devbox config will be stored locally on your machine.

Quick install

curl -fsSL "https://github.com/Tarun-Elango/devbox-cli/releases/latest/download/devbox-$(uname -s | tr '[:upper:]' '[:lower:]')-$(uname -m | sed 's/x86_64/amd64/;s/aarch64/arm64/')" -o /tmp/devbox
chmod +x /tmp/devbox
sudo install -m 755 /tmp/devbox /usr/local/bin/devbox
devbox version

Common commands

  • devbox setup — configure AWS credentials
  • devbox create <name> — create a box
  • devbox ls — list boxes
  • devbox ssh <name> — connect via SSH
